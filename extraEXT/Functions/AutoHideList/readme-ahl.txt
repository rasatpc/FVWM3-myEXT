# readme-ahl.txt

This function allows withdrawing a window to a location where it does not use precious desktop space. It can hide any window you like, read more.
https://www.fvwm.org/Wiki/Tips/AutoHidingWindows/

To install copy folder AutoHideList/ to userEXT/.

## Add the below line below "Functions".
Read $[USER_DIR]/AutoHideList/autoHideL.sys #101

Untag and add your own in autoHideL.sys
