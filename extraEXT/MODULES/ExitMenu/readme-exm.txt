# readme-exm.txt

To install or edit .fvwm/ExtraEXT.sys

## Add the below line below "Modules".
Read $[EXTRA_DIR]/MODULES/ExitMenu/exitPanel.sys #614

## Any addon or modification in .fvwm/UserEXT.sys

## Fvwm Kise

Copy .fvwm/extraEXT/MODULES/ExitMenu to .fvwm/user. Add this line in .fvwm/userExt.sys in MODULES section.
Read $[USER_DIR]/ExitMenu/exitPanel.sys

IMPORTANT:
Edit/untag where there is an Fvwm Kise label.
