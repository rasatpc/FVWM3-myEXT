To install or edit .fvwm/ExtraEXT.sys

## Add the below line below "Modules".
Read $[EXTRA_DIR]/MODULES/MyRightPanel/myRPanel.sys #606

## Any addon or modification in .fvwm/UserEXT.sys

## Fvwm Kise

Copy .fvwm/extraEXT/MODULES/MyRightPanel to .fvwm/user. Add this line in .fvwm/userExt.sys in MODULES section.
Read $[USER_DIR]/MyRightPanel/myRPanel.sys

IMPORTANT:
Edit/untag where there is an Fvwm Kise label.
