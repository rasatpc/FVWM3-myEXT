# readme-wfs.txt

To install or edit .fvwm/ExtraEXT.sys

## Add the below line below "Modules".
Read $[EXTRA_DIR]/MODULES/WifiSnd/wifiSound.sys #612

## Fvwm Kise

Copy .fvwm/extraEXT/MODULES/WifiSnd to .fvwm/user. Add this line in .fvwm/userExt.sys in MODULES section.
Read $[USER_DIR]/WifiSnd/wifiSound.sys

IMPORTANT:
Edit/untag where there is an Fvwm Kise label.
