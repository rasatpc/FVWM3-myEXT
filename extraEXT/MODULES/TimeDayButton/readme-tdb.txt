# readme-tdb.txt

To install or edit .fvwm/ExtraEXT.sys

## Add the below line below "Modules".
Read $[EXTRA_DIR]/MODULES/TimeDayButton/timeDayB.sys #609

## Any addon or modification in .fvwm/UserEXT.sys

## Fvwm Kise

Copy .fvwm/extraEXT/MODULES/TimeDayButton to .fvwm/user. Add this line in .fvwm/userExt.sys in MODULES section.
Read $[USER_DIR]/TimeDayButton/timeDayB.sys

IMPORTANT:
Edit/untag where there is an Fvwm Kise label.
