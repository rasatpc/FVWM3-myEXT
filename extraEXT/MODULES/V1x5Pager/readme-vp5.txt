# readme-vp5.txt

To install or edit .fvwm/ExtraEXT.sys

## Add the below line below "Modules".
Read $[EXTRA_DIR]/MODULES/V1x5Pager/vPager5.sys #610

## Any addon or modification in .fvwm/UserEXT.sys

## Fvwm Kise

Copy .fvwm/extraEXT/MODULES/V1x5Pager to .fvwm/user. Add this line in .fvwm/userExt.sys in MODULES section.
Read $[USER_DIR]/V1x5Pager/vPager5.sys

IMPORTANT:
Edit/untag where there is an Fvwm Kise label.
