# readme-dtn.txt

You will need x11-apps for this style to work. It takes a snapshot of the window before it is iconified, and then assigns the icon to the iconified window. This minimized style uses two configs. The second one is from addy-fe's Diary theme.

Package dependency Styles/FvwmThumbnails

To install or edit .fvwm/ExtraEXT.sys

## Add the below lines below "Styles".
Read $[EXTRA_DIR]/Styles/DiaryThumbnails/diaryThumbPlace.sys #202
